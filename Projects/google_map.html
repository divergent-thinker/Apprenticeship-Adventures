Google maps

I embedded the below map using Google's map API.

You can find documentation for the Google map API using Javascript. There are different APIs for different technologies.

To get started, you first need to add the script tag for the Google maps API: script tag src="http://maps.googleapis.com/maps/api/js">script tag

To add parameters such as the key, put a '?', the name of the parameter, in this case 'key' then '=' followed by the value of the key. e.g.http://maps.googleapis.com/maps/api/js?key=AIzaSyAFjF79SXGdZcNM4yrNGdBhF1nd9nVhe2M>
