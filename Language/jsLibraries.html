Javascript libraries are code that other people have written so you don't have to. There are different types of library...

General purpose libraries

These include:

  • Google's Closure Library
  • Mootools
  • Yahoo YUI library

They have a lot of functions to help you navigate through the DOM. "They include code for cross-browser detection, so it's easier to work with events without worrying which version of Internet Explorer your user might have, or working with animation. They make it easy to have your JavaScript request information from a web server, then update the page when it gets a response, something often referred to as AJAX."

To use these libraries, just download or link to them.

Specialised libraries

These include things like:

  • Lightbox
  • Libraries for creating slideshows
  • Drag and drop and working with accordians
  • Library to help you create curvy corners on divs.

These are mostly open source and free

JQuery

By far the most popular Javascript library is JQuery.
