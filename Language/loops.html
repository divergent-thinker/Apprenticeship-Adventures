The below explains the basic structure of a loop, and a couple of different ways of writing loops. First we begin with the basic example, and then we will look at an abbreviated version which is much more commonly used.

A repeat loop is a control structure. Another type of control structure is a conditional statement. That looks like this:

if (boolean expression) {
conditional statement
}

If the boolean expression is TRUE then the conditional statement will executed ONCE, and then we go to the next line of code underneath.

The control structure of a loop is written just like this but with a different key word. There are two ways of doing this. The first way is as follows

  1. while (boolean expression) { conditional statement
    }

If the boolean expression is TRUE, then the conditional statement will be executed MULTIPLE times until it is not true, then it will go to the next line of code.

The danger is that your program will get stuck inside this loop.
