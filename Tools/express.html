What is the Assert module?

Documentation for the assert module can be found here: Assert module documentation.

This module is used so that Node.js can test itself. Basically, it tests if a value is truthy. With assert, if you assert something and it's true nothing will happen, but if you assert something that is false it will throw an error.

How to call a module in Node

Require is a function available in Node. When you call require and pass in a module, it will make that module available. To make the assert module available for example, you would write: var assert = require('assert');

Why use Chai instead of Assert?

I've used chai instead of node's built in assert module because it's more readable than assert (see below).

In order to call the chai module in node, you do the same as with assert (see above). i.e.: var chai = require('chai');

Language options in Chai

There are various options with chai that you can select depending on your personal preferences. You can use 'should', 'expect' or 'assert'. My personal preference is 'expect' so this variable is setting that preference: var expect = chai.expect; For more information see the Chai documentation
