Handlebars

Handlebars is a templating engine which allows you to build semantic templates. It is based on Mustache template language, but improves on it in a number of key areas.

Installation

See here for Handlebars installation instructions

To install via terminal user npm install --save handlebars

Once you have installed handlebars, you will need to require it in your JS file e.g. var handlebars = require('handlebars');
