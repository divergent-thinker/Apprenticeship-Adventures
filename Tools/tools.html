Toolbelt

Git hub

Git is an open-source version control system. Git is a command-line tool, and Github is the hub for all Git activity, where developers can store their projects, network and collaborate.

iTerm

iTerm is a replacement for Terminal on Mac. It has additional features like split-pane, find-on page feature with highlighting and is much more configerable that Terminal.

Processing.JS

P5.js is a Javascript library that was created to make coding accessible to artists, designers, educators and beginners.

Ace

A code editor that you can embed on your websites. Enter this section to try it for yourself.

Homebrew

Homebrew is a free open source software package management system that simplifies the installation of software on the Mac OS X operating system. http://brew.sh/

Jekyll

Jekyll allows you to create templates in which to store information that is repeated across pages in your site. New pages can then be based on a template. This means that when it comes to updating information that appears on several pages, you can just alter it on the template instead of having to alter a number of pages.

Ajax

Ajax stands for Asynchronous JavaScript and XML. Ajax is a client-side script that allows data to be exchanged with a server. This enables a particular DOM object on the page to be refreshed, without refreshing the whole webpage.
