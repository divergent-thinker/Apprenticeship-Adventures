Mocha

Mocha is a JavaScript test framework running on node.js, featuring browser support, asynchronous testing, test coverage reports, and use of any assertion library. See Mocha Documentation

Using Mocha

To install Mocha globally, type into the terminal npm install -g mocha. Note that npm stands for 'node package manager'
